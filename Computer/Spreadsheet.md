## Answer these questions.

### 1. What is a spreadsheet?
A spreadsheet is a computer application used to organize, calculate, and analyze data in rows and columns.

### 2. Name any two spreadsheet software.
Microsoft Excel and Google Sheets.

### 3. What is a cell in a spreadsheet?
A cell is the intersection of a row and a column where data can be entered.

### 4. What is a formula in a spreadsheet?
A formula is an equation used to perform calculations on data, for example: `=A1+B1`.

### 5. What is a function in a spreadsheet?
A function is a predefined formula that performs specific calculations, such as `=SUM(A1:A5)`.
